Talesmith - 創作シナリオ管理サイト

概要

Talesmithは、自分のシナリオ作成をサポートするための管理サイトです。ユーザーがシナリオを作成・管理・共有するためのツールとして、創作活動をより効率的に行えるように設計されています。

主な特徴

  • シナリオの作成・保存
  • 登場人物や設定の管理
  • シナリオごとの進行状況の確認
  • ユーザー間でのシナリオの共有機能

使用した技術

Talesmithは、以下の技術スタックを使用して開発されました：

  • フロントエンド: React
  • バックエンド: Spring Boot
  • データベース: MySQL
  • その他: Docker, AWS RDS
